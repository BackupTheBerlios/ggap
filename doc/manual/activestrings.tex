\section{Values}

These are builtin values available to scripts in Active Strings plugin.

\variable{\_0} - full matched string.

\variable{\_1}, \variable{\_2}, ... - strings matched by whildcards in the 
pattern, if any.

%% Functions
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\section{Functions}

These are builtin functions available to scripts in Active Strings plugin.

\function{Bs}{n = 1} - equivalent to \arg{n} pressing of Backspace key.

\function{Del}{n = 1} - equivalent to \arg{n} pressing of Delete key.

\function{Ins}{arg1, ...} - inserts given strings into the document at the 
cursor position.

\function{Up}{n = 1} - moves cursor \arg{n} positions up.

\function{Down}{n = 1} - moves cursor \arg{n} positions down.

\function{Left}{n = 1} - moves cursor \arg{n} positions left.

\function{Right}{n = 1} - moves cursor \arg{n} positions right.

\function{Sel}{n} - selects \arg{n} characters starting from cursor
position. \arg{n} may be positive or negative, to indicate selection
direction.

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\section{Examples}

\begin{verbatim}
Bs(Len(_0));
\end{verbatim}
Erases matched string, aka "you can't type this string in".

\begin{verbatim}
Sel(-Len(_0)); Bs();
\end{verbatim}
Same as the previous example.
